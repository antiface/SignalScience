## Signal Processing in Python:
* [CPJKU/madmom: Python audio and music signal processing library](https://github.com/CPJKU/madmom)
* [danilobellini/audiolazy: Expressive Digital Signal Processing (DSP) package for Python](https://github.com/danilobellini/audiolazy)
* [craffel/mir_eval: Evaluation functions for music/audio information retrieval/signal processing algorithms.](https://github.com/craffel/mir_eval)
* [matousc89/padasip: Python Adaptive Signal Processing](https://github.com/matousc89/padasip)
* [r9y9/pysptk: A python wrapper for Speech Signal Processing Toolkit (SPTK).](https://github.com/r9y9/pysptk)
* [pytorch/audio: Data manipulation and transformation for audio signal processing, powered by PyTorch](https://github.com/pytorch/audio)
* [pyspace/pyspace: Signal Processing And Classification Environment in Python](https://github.com/pyspace/pyspace)
* [alexforencich/verilog-dsp: Verilog digital signal processing components](https://github.com/alexforencich/verilog-dsp)

* [jshap70/TensorFlow-Signal-Processing: doing audio digital signal processing in tensorflow to try to recreate digital audio effects](https://github.com/jshap70/TensorFlow-Signal-Processing)

* [davidpraise45/Audio-Signal-Processing: Removing background noise in a sound file](https://github.com/davidpraise45/Audio-Signal-Processing)
* [waveflow-team/waveflow: Waveflow: signal processing with tensorflow.](https://github.com/waveflow-team/waveflow)
